## Основные проблемы и решения

### 1. Заголовки и текст
**Проблема:** Слишком крупные заголовки на мобильных устройствах  
**Решение:**
- Добавлены адаптивные размеры: `text-2xl sm:text-3xl lg:text-4xl`
- Уменьшены отступы: `py-6 sm:py-10`, `gap-4 sm:gap-6`
- Добавлены `break-words` для длинных текстов

**Файлы:**
- `Dashboard.tsx`
- `PublicOrganization.tsx`
- `OrganizationPosts.tsx`
- `OrganizationReviews.tsx`
- `OrganizationQrPoster.tsx`
- `Register.tsx`
- `Login.tsx`

### 2. Кнопки
**Проблема:** Кнопки слишком маленькие для мобильных (менее 44px)  
**Решение:**
- Добавлен `min-h-[44px]` для всех интерактивных кнопок
- Адаптивные размеры текста: `text-xs sm:text-sm`
- Скрытие длинных текстов на мобильных: `<span className="hidden sm:inline">...</span>`

**Файлы:**
- `Dashboard.tsx` - все кнопки в блоке организации
- `OrganizationReviews.tsx` - кнопки фильтров и действий
- `OrganizationPosts.tsx` - кнопка "Создать пост"
- `Login.tsx` - кнопки входа
- `Register.tsx` - кнопки регистрации

### 3. Формы и фильтры
**Проблема:** Форма фильтров в каталоге организаций не помещалась на мобильных  
**Решение:**
- Изменена сетка: `sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-5`
- Поиск занимает 2 колонки на sm: `sm:col-span-2`
- Чекбокс и кнопка занимают нужное количество колонок

**Файлы:**
- `PublicOrganizationsCatalog.tsx`

### 4. Карточки и списки
**Проблема:** Карточки с контентом перекрывались на мобильных  
**Решение:**
- Изменен layout с `flex items-start justify-between` на `flex-col gap-3 sm:flex-row`
- Добавлены `min-w-0` и `flex-shrink-0` для правильного обрезания текста
- Адаптивные размеры иконок и бейджей

**Файлы:**
- `OrganizationPosts.tsx` - карточки постов
- `OrganizationReviews.tsx` - карточки отзывов
- `PublicOrganization.tsx` - бейджи и кнопки

### 5. QR Poster страница
**Проблема:** Двухколоночный layout не работал на мобильных  
**Решение:**
- Grid уже был адаптивным (`lg:grid-cols-2`), но улучшены:
  - Адаптивные размеры заголовков
  - Адаптивный размер URL input с кнопкой копирования
  - Скрытие текста "Копировать" на мобильных, только иконка

**Файлы:**
- `OrganizationQrPoster.tsx`

### 6. Таблицы и статистика
**Проблема:** Статистические карточки могли быть слишком узкими  
**Решение:**
- Grid уже был адаптивным (`md:grid-cols-3`), проверено
- Кнопки фильтров обернуты в `flex-wrap` для переноса

**Файлы:**
- `OrganizationReviews.tsx`

## Технические детали

### Использованные breakpoints
- `sm:` - 640px (планшеты портрет, большие телефоны)
- `md:` - 768px (планшеты)
- `lg:` - 1024px (планшеты альбом, десктоп)

### Основные паттерны
1. **Адаптивные размеры текста:** `text-xs sm:text-sm`, `text-2xl sm:text-3xl`
2. **Адаптивные отступы:** `py-6 sm:py-10`, `gap-4 sm:gap-6`
3. **Минимальная высота кнопок:** `min-h-[44px]` (рекомендация Apple HIG)
4. **Адаптивный layout:** `flex-col sm:flex-row`, `grid-cols-1 sm:grid-cols-2`
5. **Скрытие текста на мобильных:** `hidden sm:inline`
